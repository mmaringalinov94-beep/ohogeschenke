# README This repository contains a simple **multi-page static website** for **Ohogeschenke** (English / German). It's designed to be warm & artistic, mobile-friendly, and easy to maintain. Drop your product photos into `uploads/products/` and update `uploads/products/products.json` to add new items. --- ## File structure (all files included below) - `index.html` — Home / Startseite - `products.html` — Products / Produkte (loads `uploads/products/products.json`) - `about.html` — About / Über uns - `contact.html` — Contact / Kontakt (contact form builds a mailto: link) - `styles.css` — All styles (responsive) - `app.js` — Handles language switcher, menu, and products loading - `uploads/products/products.json` — Example product metadata (you should edit) - `assets/` — placeholder for logos or common images (you can replace) --- ohowein— Art & Wine Racks
